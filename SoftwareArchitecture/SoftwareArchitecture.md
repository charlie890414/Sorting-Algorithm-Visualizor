## UI
### Menu
1. Enter: into episode.
2. up:  cursor up.
3. down: cursor down.
4. right: cursor right.
5. left: cursor left.
Has 20 button, each button represent a episode.
each row has 4 button.
whole page has 5 rows.
pre-19 button is default episodes, and the last one is random scheme.
